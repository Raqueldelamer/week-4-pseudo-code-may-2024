Ice Cream Problem

Problem

There are 8 kids in line to get ice cream. Some are boys, some are girls. How many boys are in line?

Link to problem on Canva

Solution

For Each Person in Line 
  If the person is a boy 
  then tally
  Else do nothing

Picture/Diagram

This is my diagram from Canva.

Pseudocode

        let boysCounter = 0

        for each person in line 
        if person is a boy then 
        boysCounter = boysCounter +1
        end if
        end for

        print (boysCounter)

Discussion 

  In the line of kids, 
  look at each position that 
  each kid is in line, and the boys 
  counter's position is tallied.
  The boys counter is increased when
  there is a boy in that position in 
  line
